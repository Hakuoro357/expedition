## Определения готовности

### `Done` для gameplay-задач

- правило реализовано;
- проверено руками в нескольких партиях;
- не ломает базовый цикл;
- UI для функции читаем и понятен.

### `Done` для платформенных задач

- сценарий работает в сборке;
- есть обработка ошибок;
- нет критического сбоя при нештатном ответе SDK.

### `Done` для контента

- контент подключен в игру;
- тексты локализованы;
- награды отображаются корректно;
- нет пустых или битых ссылок на ресурсы.
