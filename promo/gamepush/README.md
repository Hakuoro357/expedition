# GamePush promo materials — структура

С версии **0.3.30** (локализация на 7 языков) промо материалы хранятся в
подпапках по локалям:

```
promo/gamepush/
  icon-1024x1024.png        ← языконезависимые иконки (общие для всех локалей)
  icon-512x384.png
  icon-800x1200.png
  ru/                       ← русская локаль
    cover-1920x1080.png     ← landscape обложка
    cover-1080x1920.png     ← portrait обложка
    screenshot-1280x720-1-prologue.png   ← 16:9 скриншоты
    screenshot-1280x720-2-map.png
    ...
    screenshot-720x1280-1-prologue.png   ← 9:16 скриншоты
    ...
  en/                       ← английская (та же структура файлов)
  tr/
  es/                       ← испанский (требуется локализация)
  pt/                       ← португальский (требуется локализация)
  de/                       ← немецкий (требуется локализация)
  fr/                       ← французский (требуется локализация)
```

## Правила

- **Внутри каждой папки локали** — одинаковый набор файлов с одинаковыми именами.
  Никаких суффиксов `-ru` / `-en` / `-tr` в именах файлов; локаль читается из папки.
- **Иконки** не локализуются (не содержат текста), поэтому лежат в корне
  `promo/gamepush/` и переиспользуются между локалями.
- **Обложки и скриншоты** локализованы — содержат UI-текст / название игры.
- При загрузке в GamePush панель выбираем папку нужной локали и заливаем
  все файлы оттуда.

## Статус локалей

| Локаль | Обложки | Скриншоты | Готовность |
|--------|---------|-----------|------------|
| ru | ✅ | ✅ | готово |
| en | ✅ | ✅ | готово |
| tr | ✅ | ✅ | готово |
| es | ❌ | ❌ | требуется локализация (v0.3.30 добавила UI, промо pending) |
| pt | ❌ | ❌ | требуется локализация |
| de | ❌ | ❌ | требуется локализация |
| fr | ❌ | ❌ | требуется локализация |

Для локалей без собственных промо — публикация в соответствующем регионе
откладывается до появления материалов. GamePush не требует промо для
конкретной локали, если игра не таргетируется на этот регион.

## Процесс добавления новой локали

1. Создать папку `promo/gamepush/<locale>/`.
2. Сделать скриншоты в игре с установленной локалью (переключить в меню).
3. Подготовить landscape и portrait обложки с локализованным названием
   (для RU — «Пасьянс Экспедиция», для EN — «Solitaire Expedition»; для
   других локалей — зависит от брендинга; при отсутствии — использовать
   англоязычное «Solitaire Expedition» как международное).
4. Положить файлы в папку с каноническими именами (см. пример `ru/`).
5. Залить в GamePush панель.
