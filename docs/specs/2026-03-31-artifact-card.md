# Solitaire: Expedition — Artifact Card

## Статус

`Draft update for approval`

## Цель

При нажатии на открытый артефакт в дневнике показывать отдельную карточку артефакта поверх текущего экрана.

## Поведение

- карточка открывается только для `открытого` артефакта;
- нажатие на закрытый артефакт ничего не делает;
- карточка показывается как overlay/modal поверх `Diary`;
- фон дневника затемняется;
- повторный переход в другую сцену не нужен;
- закрытие карточки:
- по кнопке `Закрыть`;
- по тапу вне карточки.

## Содержимое карточки

- название артефакта в верхней части карточки;
- большая картинка артефакта из папки `src/assets/artifacts/large`;
- короткое описание артефакта из `artifacts.ts` под изображением;
- компактная кнопка `Закрыть` под описанием.

## Визуальная подача

- карточка центрирована;
- размер заметно больше плитки, но не во весь экран;
- модальное окно допускается увеличить по высоте, если это нужно для более цельной композиции;
- основное внимание на изображение;
- название идёт в верхней части карточки;
- изображение увеличено и вписано в модальное окно без лишней пустоты;
- у изображения не должно быть лишних вертикальных боковых полос;
- описание остаётся под изображением;
- кнопка `Закрыть` меньше текущей версии и расположена ближе к описанию;
- стиль продолжает текущий `Diary`: тёмно-зелёный фон, светлая рамка, спокойная архивная подача.

## Источник данных

Используем уже существующие поля из `ARTIFACTS`:

- `id`
- `titleRu` / `titleEn`
- `descriptionRu` / `descriptionEn`

Для изображения добавляется связь на `large`-версию:

- `artifact_stamp_grid` -> `artifact_stamp_large`
- `artifact_map_fragment_01_grid` -> `artifact_map_fragment_01_large`
- `artifact_unidentified_object_grid` -> `artifact_unidentified_object_large`
- `artifact_levin_note_grid` -> `artifact_levin_note_large`
- `artifact_unsigned_note_grid` -> `artifact_unsigned_note_large`
- `artifact_case_grid` -> `artifact_case_large`
- `artifact_hidden_map_large_grid` -> `artifact_hidden_map_large`
- `artifact_disc_container_grid` -> `artifact_disc_container_large`
- `artifact_archive_seal_grid` -> `artifact_archive_seal_large`

## Ограничения первой итерации

- только `Diary`;
- без анимационного перехода;
- без перелистывания между артефактами;
- без отдельного fullscreen viewer;
- без изменения save-state.

## Критерий готовности

- в `Diary` открытый артефакт нажимается;
- появляется карточка с большой картинкой и описанием;
- карточка закрывается без смены сцены;
- закрытые артефакты не открывают карточку.
